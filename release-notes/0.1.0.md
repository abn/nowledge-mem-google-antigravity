# Nowledge Mem Google Antigravity Plugin v0.1.0

Bring your Nowledge Mem knowledge base into Google Antigravity with persistent context, memory rules, and agent skills.

## What's Included

- **Automatic Lifecycle Hooks**:
  - `PreInvocation` loads Context Bundle or Working Memory on startup and injects it as an ephemeral message.
  - `Stop` extracts the raw conversation messages from Antigravity's `transcript.jsonl` log and syncs them as a thread into Nowledge Mem when execution finishes.
- **MCP Servers Configuration**:
  - Exposes the local `nowledge-mem` MCP server tools at `http://127.0.0.1:14242/mcp/`.
- **Integrations and Rules**:
  - `rules/nowledge-mem.md` guides the agent on when and how to search, save, distill, and sync memories.
- **Agent Skills**:
  - `read-working-memory`
  - `search-memory`
  - `distill-memory`
  - `save-thread`
  - `save-handoff`
