# Nowledge Mem Google Antigravity Plugin v0.1.1

Introduce workspace skill management capabilities to list, recommend, and install/update skills directly in target workspaces.

## What's Included

- **Skill Management Capability**:
  - Implemented the `nmem-manage-skills` agent skill.
  - Implemented `manage_skills.py` script under `skills/nmem-manage-skills/scripts/` to communicate with the Nowledge Mem server.
- **Features**:
  - List available active, candidate, and archived skills from Nowledge Mem.
  - Suggest relevant skills based on files present in the workspace.
  - Compile candidate skills on-demand.
  - Download and install skill bodies using `include_body=true` directly without global activation side-effects.
  - Optionally exclude the installed skill folders locally in Git via `.git/info/exclude` if keeping them user-specific.
