# Nowledge Mem Google Antigravity Plugin v0.1.2

Introduce the `nmem-propose-skill` agent skill to support rich interactive drafts and proceed approvals prior to creating/updating skills on Nowledge Mem.

## What's Included

- **Propose Skill Capability**:
  - Implemented the `nmem-propose-skill` agent skill.
  - Guided creation/improvements via user-facing review drafts and interactive "Proceed" button approvals.
  - Safe creation using `create_skill` MCP tool or `propose_skill_improvement` MCP tool.
